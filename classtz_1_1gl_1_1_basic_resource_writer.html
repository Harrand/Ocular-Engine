Detailed Description

template<typename T>
class tz::gl::BasicResourceWriter< T >

Writes data to an existing block of memory.

This is likely to be some mapped tz::gl::IBuffer memory. This is used to copy objects into a block of memory one-at-a-time in an ordered and contiguous manner.

Example: Iterate through each object in the scene, using this writer to write some object data into a buffer. Afterwards, the buffer can be given to a tz::render::Device as a resource-buffer and used to render in one go.

template<typename T >
bool tz::gl::BasicResourceWriter< T >::write ( T  element)

Write the given object into the memory block.

If this is the first write, it will be written to the very beginning of the block. Otherwise, it will be written directly after the last write. Precondition: The memory block is valid and pre-allocated. Otherwise, this will invoke UB without asserting. Note: If the memory block is not large enough to hold another object, the write will be aborted and false is returned.
