Detailed Description

template<typename T, std::size_t dimension>
class Tree< T, dimension >

Represent a tree of any dimensions, holding any type. Data members remain protected without any utility functions. This is so any concrete data type (such as a binary search tree) can implement their own functionality as they should.

Template Parameters
T- Type of each node of the tree
dimension- Number of children per tree (for example, a BinaryTree is 2, a QuadTree is 4)
