constexpr

Perform a binary clamp on a value between lo and hi.

A typical clamp will take any value and return a value between lo and hi. A binary clamp will instead take any value between lo and hi and return a value equal to either lo or hi, depending on which one is closer to the value. Example: binary_clamp(5, 10, 7, Lower) yields 5. 5 is closer to 7 than 7 is to 10. If the Higher bound is instead used, the returned value will be 10.

Template Parameters
TUnderlying value type to use.
Parameters
loLowest value which could be returned.
hiHighest value which could be returned.
valValue to saturate.
boundChosen behaviour if the value is equidistant between lo and hi.
Returns
If val is closer to lo, returns lo. If val was closer to hi, returns hi.

This clamp will take a value and a factor, and return a multiple of factor that was closest to val.

Example: clamp_by_factor(8, 10, Lower) yields 8. 10 is between the multiples 8 and 16, but is closer to 8. If the Higher bound is instead used, 16 is returned.

Template Parameters
TUnderlying value type to use.
Parameters
factorFactor which the value will saturate to.
valValue to saturate.
boundChosen behaviour if the value is equidistant between two multiples.
Returns
A multiple of 'factor' which 'val' is closest to.
