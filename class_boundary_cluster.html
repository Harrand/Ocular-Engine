◆ ClusterIntegration

enum BoundaryCluster::ClusterIntegration
strong

The method in which the current cluster element is integrated with the remainder of the cluster. UNION means that it can be collided with either the remainder of the cluster, or this new section. INTERSECTION means that it can be collided with only if both the remainder and this new section are collided with.
