Detailed Description

template<class T>
class Uniform< T >

Represent an OpenGL uniform in C++. Supports the following Topaz/C++ primitives: bool, int, unsigned int, float, double, Vector2F, Vector3F, Vector4F, Matrix2x2, Matrix3x3, and Matrix4x4. If the template argument is not any of these types, a static assertation will fail in Uniform<T>::push and emit a compiler error.
