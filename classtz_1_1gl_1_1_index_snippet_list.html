Detailed Description

IndexSnippetLists contain ranges of indices, where each index is an offset into an existing index-buffer.

In other words, these indices are indices into the index-buffers indices. Unfortunately the inner-workings are a bit complex. For clarity, see the topaz_multi_draw_demo's code as a useful example.

Note: This snippet need not be aware of the Object corresponding to the ibo_index. This is confusing but does increase flexibility of this object. Precondition: The index must be valid in the eyes of that object with which this snippet will be rendered with.
