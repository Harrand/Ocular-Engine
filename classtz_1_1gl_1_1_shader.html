Detailed Description

Shader components which are attached to ShaderPrograms.

Shaders can be one of several types, indicated by the ShaderType enum.

Shaders are written in GLSL. The shader source can be uploaded to this Shader component. Once that's done, compilation can be done. Compilation of a Shader requires an existing ShaderCompiler. Some compilers do the bare-minimum, others may include useful preprocessing directives (TZGLP).
