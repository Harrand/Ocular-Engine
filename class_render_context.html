Detailed Description

Provides a graphical context to a scene when issuing draw calls. This structure tells the scene with which shaders to issue draw calls with. Each RenderContext must have exactly one object shader, and optionally a second shader for sprite rendering. Remark: If a sprite-shader is not provided, topaz will NOT attempt to use the object shader to render the sprites. Instead, rendering the sprites shall be skipped for this draw call. If one does not wish to render sprites, simply omit the sprite shader in the given RenderContext.
