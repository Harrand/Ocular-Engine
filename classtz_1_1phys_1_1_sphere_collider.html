CollisionPoint tz::phys::SphereCollider::test_against ( const ICollider &  rhs) const
virtual

Test this collider against another existing collider.

Precondition: The collider types must be compatible. If not, this will assert and invoke UB. Precondition: Both colliders must have been push()ed so that they are both in world-space. Otherwise this will yield unexpected results without asserting.
