bool tz::render::MeshAsset::shares ( tz::gl::Object *&  object,
std::size_t &  ibo 
) const

Query as to whether this asset shares the given gl::Object and IBO handle.

If any of them don't match, they will be assigned to this MeshAsset's Object/IBO. Note: It is not an error to pass nullptr/an invalid IBO handle in this case. In this case they will simply be assigned to match this asset.

Parameters
objectReference to the object pointer to compare.
iboReference to the IBO handle to compare.
Returns
True if the object pointer & IBOs match and no assignment took place. Otherwise false.
